## 画像

コンテンツ内に表示する画像は、2種類あります。（og画像やファビコンなどは含みません）<br/>

|                    | src画像                             | public画像            |
| ------------------ | ----------------------------------- | --------------------- |
| タグ               | Imageコンポーネント（astro:assets） | imgタグ               |
| 最適化             | される                              | されない              |
| webp               | 自動で変換                          | 自分で変換            |
| サイズ指定         | 自動                                | 手動                  |
| 画像名             | Astroが命名（hash付き）             | 自分で命名            |
| ファイルを置く場所 | /src/image/                         | /public/assets/image/ |

### アートディレクション画像

ブレイクポイントで画像の縦横比が変わる画像をアートディレクション画像と呼びます。<br/>
Astroのg亜増最適化を使用しないpublic画像を使用する場合は、従来のWebサイトと同じように picture タグを使用します。

src画像を使用するために、専用のコンポーネント（ArtDirectionImage）を用意しています。propsで必要な画像を指定するだけで、自動で画像は最適化され必要な値を設定できます。

### 備考

- svg画像は、publicフォルダに置いた上でimgタグを使用して配置します。
- Imageコンポーネントからpublicフォルダにある画像を指定できるのですが、最適化できない上、widthやheightを自分で入力する必要があるので、検討の対象がいとしています。
- Astroのドキュメントは[コチラ](https://docs.astro.build/ja/guides/images/#astro%E3%83%95%E3%82%A1%E3%82%A4%E3%83%AB%E5%86%85%E3%81%AE%E7%94%BB%E5%83%8F)。
